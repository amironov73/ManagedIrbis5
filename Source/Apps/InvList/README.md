### InvList

Простая утилита для построения списка экземпляров.

Командная строка:

```
InvList <connectionString> <searchExpression>
```

здесь

* **connectionString** - строка подключения к серверу ИРБИС64;
* **searchExpression** - выражение для поиска по словарю.

Пример:

```
InvList "host=192.168.1.3;user=librarian;password=secret;db=IBIS;" "MHR=Ф603" > fond603.txt
```
