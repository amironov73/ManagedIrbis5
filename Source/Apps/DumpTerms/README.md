### DumpTerms

Утилита для дампа терминов поискового словаря.

```
DumpTerms <connectionString> <prefix>
```

* **connectionString** -- строка подключения.
* **prefix** -- префикс терминов.

Пример:

```
DumpTerms "host=192.168.3.4;user=admin;password=hello;db=IBIS;" "J="
```
